提供了C++和MATLAB代码做TTI介质的波场模拟。

C++ and MATLAB code are provided for wavefield modeling in TTI media.

你可以直接用g++编译c++代码，并使用./运行代码。 

You can compile the c++ code directly using g++ and run the code with ./ .
